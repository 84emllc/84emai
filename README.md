# 84EM.AI

Personal AI resource site sharing workflows, discoveries, tool reviews, and practical applications.

**Live site:** https://84em.ai

## Quick Start

```bash
# Development server
hugo server -D

# Build for production
hugo
```

## About

Built with Hugo. No JavaScript, no frameworks, no external dependencies. System fonts, single CSS file, WCAG 2.2 AA accessible.

By [Andrew Miller](https://84em.com).
